# tribe_marketplace
Command line application

## Prerequisites

* Ruby 2.6.3

## Testing

To install gems run:

    $ bundle install

To run tests execute:

    $ rspec --format doc

Then open code coverage report in the browser:

    $ open coverage/index.html

To run the solution execute with sample data:

    $ ruby solution.rb 10 IMG 15 FLAC 13 VID
